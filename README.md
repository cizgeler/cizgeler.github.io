# Çizgeler

Çizgeler, matematiğin türlü durumlarını, aralarındaki ilişkileri de gözeterek animasyonlarla göstermeye çalışan  bir [Youtube  kanalı](https://www.youtube.com/channel/UCgDlaO17Z7me_R2uaus_o_A)dır. [3blue1brown](https://www.youtube.com/3blue1brown) kanalından ilham alır. 


 Bu web sitesinde, video içeriklerini zenginleştirmek ve daha etkileşimli hâle getirmek  için  tamamlayıcı etkinliklere yer verilir.
